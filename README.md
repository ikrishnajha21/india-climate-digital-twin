INDIATWIN is a full-stack, browser-based Climate Digital Twin platform built in direct response to the above 
problem statement. It presents a "mission control" style interface — dark, high-contrast, satellite-telemetry 
themed — through which a user can define a region of interest, walk through a simulated data-ingestion 
and modelling pipeline, explore live geospatial climate indicators, run prognostic forecasts, and stress-test 
what-if climate scenarios, all without leaving a single page application. 
The platform is structured as six cooperating subsystems: the Workflow Pipeline, the CommandCenter 
Dashboard, the 3D Globe Teaser, the Prediction Engine, the What-If Simulator, and an AI Copilot with 
Mission Logs. Each subsystem maps directly onto a stage of the data-driven digital-twin lifecycle described in 
the problem statement — definition, ingestion, preprocessing, modelling, visualisation, and simulation.
